## 🎯 项目愿景

一款可在任意 Windows 应用程序中使用的本地 AI 代码/文本补全工具，包含：
- **桌面补全服务**: 全局钩子 + 幽灵文本悬浮窗
- **Web 控制台**: 7891 端口，模型管理/配置/监控
- **记忆系统**: 持久化交互历史，增强上下文质量
- **多引擎支持**: Transformer / ExLlamaV2 / 云端 API
